# Qu::Pcr

A package for PCR, including primers, products, primer3 parser, virtual electrophoresis, etc.

## Installation

Add this line to your application's Gemfile:

    gem 'qu-pcr'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install qu-pcr

## Usage

mobility = Qu::Pcr::VirtualGel.cal_mobility(size)
